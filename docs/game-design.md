# 用宗シラス漁ものがたり — ゲーム設計書（v2: 運搬→自動化→拡張）

最終更新: 2026-06-09 / ステータス: **設計合意フェーズ（実装前）**

## コンセプト
用宗しらすの実流通「漁→水揚げ→加工→販売」に沿った、スマホ縦持ちの
**放置経営ゲーム**。手運びの動線を体で覚え、コンベア→トラック→NPC→全自動と
**一個ずつ楽にしていく**のが気持ちよさの核。

## マップ構造（縦スクロール）
```
⭐ 大漁場（大型船Lv3で解放／大きな群れ＝2倍）
🌊 海（通常）＋🚢 漁船
──── 岸壁 ────
🪣 水揚げ場（生しらすバッファ）
🏭 加工場（生→加工しらす）
   ├─ 🏪 直売所（パック／釜揚げ：安い・回転速い）
   └─ 🍚 食堂（しらす丼など：単価高い・調理あり）
👨‍👩‍👧‍👦 お客さん → 💰 売上
```

## ノードと品物
| ノード | 入力 | 出力/役割 | 補足 |
|--------|------|-----------|------|
| 海 | — | 生しらす(raw) | 重なって漁。near=+1 / 大漁場=+2 |
| 水揚げ場 | raw | rawを貯める | バッファ。容量UP可 |
| 加工場 | raw | 加工しらす(proc) | 加工速度・容量UP可 |
| 直売所 | proc | パック販売 | 低単価・客の回転速い |
| 食堂 | proc | 丼に調理して販売 | 高単価・調理時間あり |

- プレイヤーの手持ちは1種類ずつ（raw / proc）。販売はお客さんが在庫から購入。

## フロー＝リンク（手運び⇄自動の切替）
| リンク | 区間 | 初期 | 自動化する設備 |
|--------|------|------|----------------|
| A | 海→水揚げ場 | 手運び | 漁船の自動操業（終盤 Lv5） |
| B | 水揚げ場→加工場 | 手運び | ベルトコンベア（Lv2） |
| C | 加工場→直売所 | 手運び | 軽トラック（Lv3） |
| D | 加工場→食堂 | 手運び | 軽トラック（Lv3） |
| 販売 | 直売所/食堂→客 | 自動(客が来る) | 販売員NPC・集客で加速（Lv4） |

## 自動化レベル（解放順・価格は**たたき台**）
1. **Lv1 手運び**：A・B・C・D 全部プレイヤー（完全移動ゲー）
2. **Lv2 ベルトコンベア**（B自動）— 約 ¥2,000 ／以降レベルで搬送速度UP
3. **Lv3 軽トラック**（C・D自動）— 約 ¥8,000
4. **Lv4 販売員NPC＋集客**（販売加速・客増・売上倍率）— 約 ¥5,000
5. **Lv5 完全自動化**（A=漁船自動操業）— 約 ¥30,000

※ 別軸で各拠点の「速度・容量」レベルUPと、漁船サイズ（大漁場解放/積載/速度）も継続。

## 2系統の販売（客単価の幅）
| 店 | 商品 | 価格(たたき台) | 特徴 |
|----|------|----------------|------|
| 直売所 | パックしらす / 釜揚げしらす | ¥120〜 | 安い・客の回転速い・在庫から即売 |
| 食堂 | しらす丼 / 生しらす丼 / 沖漬け丼 | ¥350〜 | 高単価・調理時間あり・席数（同時提供数）up可 |

- **食堂は中盤で解放**（約 ¥3,000）。序盤は直売所だけ→拡張のワクワク。
- （任意・後回し）生しらす丼は「鮮度（評判）」で単価変動、などの深み。

## オフライン収益（採用：自動化レベル依存）
- 留守中も、**自動化済みのリンクだけ**が稼働して在庫を流す。
- 収益レート ≈ 「全自動でつながっている区間の最小スループット × 売値 × 効率(50%)」。
- **手運びが残る区間で停止**（そこがボトルネック）→ 自動化するほどオフライン収益が伸びる＝自動化の動機。
- 上限：最大8時間ぶんでキャップ（青天井防止）。
- 復帰時に「🛌 留守中に ◯◯円 稼ぎました！」モーダル。

## 数値バランス（初期たたき台・後で調整）
- 漁獲：near +1/匹、大漁場 +2/匹。船Lvで移動速度・積載量・群れ数UP。
- 加工：proc 生成 1.2+0.8×Lv /秒、容量 12+6×Lv。
- 直売所：客が ¥120×(1+0.3×Lv) で購入、回転は客数依存。
- 食堂：proc→丼 0.8+0.5×Lv /秒、丼 ¥350×(1+0.25×Lv)。
- 自動搬送：各リンク 2+1×Lv 個/秒（容量でキャップ）。
