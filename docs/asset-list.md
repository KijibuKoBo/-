# 必要画像リスト（キレイなアプリ用）

最終更新: 2026-06-09 / 用途: ChatGPT等で生成→`game/assets` にアップ→組み込み

## 共通ルール（全画像）
- **画風を統一**：2頭身ちびキャラ＆ミニチュア風／温かいセル＋淡い水彩の手描きアニメ調／
  均一な茶系の輪郭線／柔らかい光／用宗漁港の世界観。**全部同じ画風・同じ光源・同じ彩度**。
- **背景は完全透過PNG**（※「テクスチャ」だけは全面塗りでOK）。
- 文字・ロゴ・枠は入れない（看板の中の文字はOK）。被写体は中央、同種は同じ縮尺。
- 長辺 768〜1024px 程度で。こちらで縮小・最適化します。
- **複数コマ**は「1枚に横一列・等間隔・同じ大きさ／コマ間でキャラがズレない」。
- 既にあるものは作り直し不要（下表「既存」）。

凡例：★必須 / ☆推奨 / ・任意
