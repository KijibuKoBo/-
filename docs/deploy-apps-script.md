# Google Apps Script デプロイ手順

このアプリのメール送信機能は、Google Apps Script の Web アプリとして
`kijibu.kobo@gmail.com` のアカウント上で動かします。

以下の手順は **初回のみ** 必要です。一度設定すれば、後はフロントエンドからの
fetch で自動的に Gmail が送信されます。

---

## 1. Apps Script プロジェクトを作る

1. `kijibu.kobo@gmail.com` で Google にログイン
2. [https://script.google.com/](https://script.google.com/) を開く
3. 左上 **「新しいプロジェクト」** をクリック
4. 左上のプロジェクト名（「無題のプロジェクト」）をクリックして
   **「発注書送信」** などに変更

## 2. コードを貼り付ける

1. デフォルトで開いている `コード.gs` を全選択して削除
2. このリポジトリの **`apps-script/Code.gs`** の中身を全部コピーして貼り付け
3. 左メニューの ⚙ **「プロジェクトの設定」**
4. 「`appsscript.json` マニフェスト ファイルをエディタで表示する」**ON**
5. エディタに戻ると `appsscript.json` が表示されるので、
   このリポジトリの **`apps-script/appsscript.json`** の中身に置き換え
6. 💾 保存（Ctrl + S）

## 3. Web アプリとしてデプロイ

1. 右上 **「デプロイ」 → 「新しいデプロイ」**
2. 歯車 ⚙ → **「ウェブアプリ」** を選択
3. 設定：
   - 説明: `発注書送信 v1`（任意）
   - 実行するユーザー: **自分 (kijibu.kobo@gmail.com)**
   - アクセス権: **全員** （← 重要：これで認証なしでフロントから叩ける）
4. **「デプロイ」** をクリック
5. 初回は **「アクセスを承認」** を求められる
   - 「詳細」 → 「（プロジェクト名）に移動」 → 「許可」
   - Gmail 送信権限を許可（あなた自身のアカウントなので安全）
6. 表示された **「ウェブアプリ URL」** をコピー
   - 形式: `https://script.google.com/macros/s/AKfycb.../exec`

## 4. フロントエンドに URL を登録

1. このアプリ（`index.html`）をブラウザで開く
2. 右下の **「⚙ 設定（初回のみ）」** を開く
3. コピーした URL を貼り付けて **「保存」**
   - ブラウザの localStorage に保存される（このブラウザでのみ有効）

## 5. テスト送信

1. フォームに適当な値を入力（送信先メールは自分のアドレスでテスト）
2. **「メール送信」** ボタン
3. メールボックスを確認。PDF 添付の発注書が届けばOK

## 自宅 PC でも使えるようにする方法

### 方法 A: ブラウザでファイルを開いて、もう一度設定する
- リポジトリを `git clone` → `web/index.html` をダブルクリック
- 右下「⚙ 設定」に同じ Apps Script URL を貼り付け
- これで自宅PCのブラウザでも送信可能

### 方法 B: GitHub Pages で公開して URL アクセス（おすすめ）
1. GitHub のリポジトリページ → Settings → Pages
2. Source: `Deploy from a branch`, Branch: `main` / Folder: `/web`
3. 保存。数分後に `https://kijibukobo.github.io/-/` のような URL で公開
4. このURLをブックマークすれば、どのデバイスからでもアクセス可能

## トラブル対処

- **送信ボタンを押すとエラー**: `⚙ 設定` の URL が間違っている可能性。
  ブラウザの開発者ツール（F12）の Network タブで失敗内容を確認。
- **「アクセスが拒否されました」**: Apps Script のデプロイ設定で
  「アクセス権: 全員」になっていない。デプロイをやり直す。
- **メールが届かない**: Gmail の「送信済み」を確認。送信されているなら
  受信側のスパム判定の可能性。
- **PDF の文字化け**: Apps Script の HTML→PDF 変換は日本語OKだが、
  特殊な記号は化けることがある。`Code.gs` の `wrapPrintableHtml` を調整。

## コード更新時

1. このリポジトリの `Code.gs` を編集 → push
2. Apps Script のエディタに**もう一度コピペ**して 💾保存
3. **「デプロイ」 → 「デプロイを管理」 → 鉛筆アイコン**
4. 「バージョン」を「新バージョン」にして「デプロイ」
5. URL は変わらないのでフロントの設定はそのまま
