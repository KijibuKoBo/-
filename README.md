# カワサキ宛 発注書アプリ

㈲松永工房（木地部）から株式会社カワサキへの発注書をブラウザで作成し、
Gmailから自動送信するWebアプリです。

## 使い方（公開後）

1. ブラウザで公開URLを開く
2. 製品テンプレートを選択（または手入力）
3. 数量・日付・備考を入力。必要なら図解画像をアップロード
4. プレビュー確認 → 「送信」ボタン
5. カワサキ宛にPDF添付メールが届く
