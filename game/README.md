# 🎣 用宗シラス漁ものがたり

用宗漁港を舞台にした、ブラウザで遊べる **シラス漁 経営シミュレーションゲーム** です。
スマホ・PC どちらでも、指（タッチ）またはマウスでキャラを動かして遊べます。

## 遊び方

1. 画面をタッチ／クリックすると、漁師キャラがその場所へ移動します。
2. キャラが拠点に入ると、その拠点の作業が自動で進みます。

| 拠点 | できること |
|------|-----------|
| 🌊 海（シラス漁） | 船の積載量いっぱいまでシラスを獲る |
| ♨️ 加工場 | 生シラス → 釜揚げシラスに加工 |
| 🍳 調理場 | 釜揚げ → シラス丼などの料理に |
| 🏪 販売所 | お客さんに売ってお金を稼ぐ |

3. 貯めたお金で **🛠 アップグレード** から成長させます。

| 強化 | 効果 |
|------|------|
| 🚤 漁船 | 漁獲スピード・積載量アップ |
| ♨️ 加工機 | 加工スピード・保管量アップ |
| 🍚 メニュー開発 | 売値アップ（シラス丼→特上海鮮丼…） |
| ⚓ 港の拡張 | お客さん増加・保管量アップ |
| 🧑‍🍳 従業員 | いない拠点も自動で稼働（加工→調理→販売の順に自動化） |

### ⚠️ マイナスイベント
- 🔧 加工機の故障（修理費 or 自力修理）
- 🌊 不漁（漁獲ダウン）
- 😣 クレーム（評判ダウン、謝罪で回復）

### 🎉 プラスイベント
- 🐟 大漁の群れ
- 🚌 観光バス到着（お客急増）

## 動かし方（ローカル）

`game/` フォルダ内の `index.html` をブラウザで開くだけで動きます。
ローカルサーバーで開く場合：

```bash
cd game
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 公開（GitHub Pages）

`game/` フォルダを GitHub Pages の公開対象にすれば、同じURLで全デバイスから遊べます。
（リポジトリ Settings → Pages → Branch を選択して `/game` を指定）

## 技術

- 依存ライブラリなし。HTML + CSS + Canvas (Vanilla JS) のみ。
- セーブは `localStorage` に自動保存されます。
