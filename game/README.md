# 🎣 用宗シラス漁ものがたり

用宗漁港を舞台にした、ブラウザで遊べる **シラス漁 経営シミュレーションゲーム** です。
スマホ・PC どちらでも、指（タッチ）またはマウスでキャラを動かして遊べます。

## 遊び方（運搬型・縦長）

1. 画面をタッチ／ドラッグして、漁師を動かします。
2. **🌊 上＝海** へ行き、泳ぐ魚に重なると **生シラス（バケツ）** が手持ちに積み上がります。
3. 品物を拠点へ運ぶと、立つだけで **自動で荷降ろし／完成品の積み込み** をします。

| 拠点 | 位置 | 流れ |
|------|------|------|
| 🌊 海 | 上 | 魚に重なって生シラスを獲る（船Lvで群れ増・速度UP） |
| ♨️ 加工場 | 左上 | 生シラス → パックに加工して積み上がる |
| 🍳 調理場 | 左下 | パック → シラス丼に |
| 🏪 販売所 | 中央 | 丼を運ぶと、並ぶお客さんが購入してお金に |

4. 貯めたお金で **🛠 アップグレード** から各拠点を成長させます。

| 強化 | 効果 |
|------|------|
| 🚤 漁船 | 移動スピード・積載量アップ／魚の群れが増える |
| ♨️ 加工場 | 加工スピード・保管量アップ |
| 🍳 調理場 | 調理スピード・保管量アップ |
| 🏪 販売所 | お客さん増加・丼の売値アップ |

### ⚠️ マイナスイベント
- 🔧 加工機の故障（修理費 or 自力修理）
- 🌊 不漁（漁獲ダウン）
- 😣 クレーム（評判ダウン、謝罪で回復）

### 🎉 プラスイベント
- 🐟 大漁の群れ
- 🚌 観光バス到着（お客急増）

## 動かし方（ローカル）

`game/` フォルダ内の `index.html` をブラウザで開くだけで動きます。
ローカルサーバーで開く場合：

```bash
cd game
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 公開（GitHub Pages）

`game/` フォルダを GitHub Pages の公開対象にすれば、同じURLで全デバイスから遊べます。
（リポジトリ Settings → Pages → Branch を選択して `/game` を指定）

## 技術

- 依存ライブラリなし。HTML + CSS + Canvas (Vanilla JS) のみ。
- セーブは `localStorage` に自動保存されます。
