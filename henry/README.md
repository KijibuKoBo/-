# ハリネズミのヘンリーさん 🦔

小さな骨董と、世界でいちばん優しい鑑定家の物語。
ハリネズミの **ヘンリーさん** が、陽の差す工房で小さな骨董を鑑定する
**4コマ絵本** を、1ページのLP（ランディングページ）にまとめたものです。

> 第一話「**プティカウチ**」――ドールサイズのカウチに宿る、縮まないエレガンス。

---

## 🎨 デザインの考え方

ヘンリーさんの世界観（アンティーク／博物図譜風／温かなセピア）を、
そのまま web の編集デザインに翻訳しています。

- **配色**：経年した生成り紙 × 朱色（カウチ）× アンティークゴールド × 漆黒
- **書体**：見出し・本文に明朝（Shippori Mincho）、欧文に Cormorant Garamond
- **質感**：紙のきめ（ノイズ）、金の飾り罫、コーナー装飾、ドロップキャップ
- **構成**：絵本を「鑑定帖」に見立て、序 → 4コマ → 細部 → 蒐集品カード で展開
- スクロールで静かに浮かび上がる演出（`prefers-reduced-motion` 対応）
- PC・スマホ両対応（レスポンシブ）

## 🖼 もっと良くするための画像（あると嬉しい素材）

現状は1枚の4コマ原画だけで構成しています。以下があると、世界観をさらに広げられます。

1. **ヘンリーさん単体の立ち絵（背景透過 / 余白あり）**
   … ヒーローの主役として大きく置けます（緑のジャケット姿の全身が理想）。
2. **工房の引きの背景イラスト（横長）**
   … ヒーローやセクションの背景に敷いて、奥行きを出せます。
3. **次の鑑定品の4コマ（第二話以降）**
   … 「つづく」を実際のエピソード一覧／カルーセルに育てられます。
4. **ヘンリーさんの小物（虫眼鏡・帽子・鑑定札など）の単体イラスト**
   … 見出しの装飾やアイコンに使えます。
5. **ロゴ／題字（手描き風の「ヘンリーさん」）**
   … あればヒーローの題字を画像に差し替えて、より絵本らしくできます。

画像は `henry/web/assets/` に置けば、こちらでレイアウトに組み込みます。
（縦横比・透過の有無だけ教えてもらえれば調整します）
