# カワサキ宛 発注書 作成・送信アプリ

このリポジトリは、㈲松永工房（GitHub上は「木地部工房 / KijibuKoBo」）が
取引先「株式会社 カワサキ」へ送る **発注書** をブラウザで作成し、
Gmail から自動送信する Web アプリです。

---

## 🎯 ゴール

- ブラウザ上のフォームに製品名・サイズ・数量・備考などを入力
- 「PDFと同じレイアウト」のプレビューを表示
- 図解は画像アップロードで貼り付け
- 「送信」ボタンで `kijibu.kobo@gmail.com` からカワサキ宛にメール送信
  （本文 + PDF 添付）
- PC・スマホ・自宅PCすべてのブラウザから同じURLで使える

## 🏗 アーキテクチャ

```
[ブラウザ (HTML/JS)]  ──fetch──▶  [Google Apps Script Webアプリ]  ──▶  [Gmail送信]
       ↑                                       ↑
   GitHub Pages 公開                  script.google.com にデプロイ
   (web/ フォルダ)                     (apps-script/ フォルダのコードを貼付)
```

なぜこの構成？
- **サーバー・課金不要**：Apps Script は Google が無料ホスト
- **Gmail送信が1行**：`GmailApp.sendEmail()`
- **アプリパスワード不要**：Googleアカウントの認可だけで送信
- **どのデバイスからも同一URL**
