# 長岡きのこ採集記 🍄

新潟県・長岡市の山を **約20年** 歩いて記録した、石原 巖（いしはら いわお）の
きのこ採集アーカイブです。

> **アプリで記録 → ホームページで公開 → 書籍化** までを一本でつなぐプロジェクト。

採集したきのこを「おしゃれで格好いい図鑑」としてブラウザで管理し、
そのままWebサイトとして公開でき、最終的には印刷・書籍化できるデータ構造で
記録を残します。

---

## ✨ いまできること（フェーズ1：アプリ）

`kinoko/web/` をブラウザで開くと——

- 🖼 **ギャラリー表示**：採集したきのこをカードで一覧
- 🔍 **検索・絞り込み**：和名／学名／場所／環境で検索、食毒（食用・毒・食不適）で絞り込み
- ↕️ **並び替え**：採集日・五十音・おすすめ度
- 📖 **詳細表示**：採集日／場所／標高／発生環境／共生樹／味・食べ方／覚え書き／似た毒きのこへの注意
- 📚 **図鑑モード**：1種1ページの図鑑レイアウトに切り替え（→ そのまま印刷・PDF化＝書籍化の下地）
- 📱 PC・スマホ・自宅PC、どのブラウザでも同じURLで

データは [`web/data/records.json`](web/data/records.json) に入っています。
いまはサンプルの長岡周辺の種（ナメコ・マイタケ・ムキタケ 等）が入っているので、
**石原巖さんの実際のノート・写真に差し替え**ていきます。

## 🚀 ローカルで見る

```bash
cd kinoko/web
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

> ※ `index.html` をダブルクリックで直接開くと、ブラウザの制限で
> `records.json` を読めないことがあります。上の簡易サーバー経由が確実です。

## ⚠️ きのこ採集の注意

食毒の最終判断は必ずご自身の責任で。少しでも不安なきのこは絶対に食べないでください。
本アプリの食毒表示・似た毒きのこの記載は、石原巖さんの経験に基づく覚え書きであり、
鑑定を保証するものではありません。
